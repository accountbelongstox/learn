MySpy是一个比Spy++功能强大的工具，基本上，Spy++有的功能MySpy都有，而MySpy有的功能，Spy++大部分都没有。
软件为绿色软件，不需要安装，解压缩后放倒任何你喜欢的文件夹，它都可以工作。
你可以取得任何"***"里的密码，支持NT/2000/XP/2003/HTML，如果不能从"***"中取得密码，不是MySpy的设计问题，而是那个密码被加密了，既是其他工具也无法得到。
系统定义一个热键Ctrl + F11，用这个热键你可以，取得密码，抓取窗体，获得窗体的位图，而你所做的就是按热键，MySpy会自动处理热键对应的功能。
想知道WindowsXp的开始菜单用什么控件做的吗？在Normal页，按下热键，移动鼠标去点击开始按钮，移动鼠标到开始菜单，看见了吗？
我获得密码后想复制怎么办？你可以在获得密码后，按热键结束获取密码模式，这样当前的密码就被保存在文本框里了。
我想使用以前MySpy的XP风格界面怎么办？打开config.ini文件，修改UseXPStyle为1即可。
我想使用以前MySpy的配色怎么办？打开config.ini文件，修改UseMySpyColor为1即可。
对于一般Spy工具是没有办法得到菜单等及时消失的窗体，但本软件可以，你可以先按热键，这是你不需要拖动鼠标，只需要移动鼠标到你想要的任何窗体、菜单上，就可以捕获这个菜单了。
操作过程同上，将软件切换到Screen页，按"Capture Window Bmp"按钮可以得到窗体位图，或者将鼠标移动到目标窗体（同样支持菜单等及时消失的窗体）按热键，就可以得到窗体的位图了。
为什么在我的计算机上无法使用MySpy？本软件仅支持NT、2000、XP及以上操作系统。
为什么我无法使用WebSpy？本软件使用了微软的OLEACC.DLL文件，而这个文件不随操作系统一同发布，所以你需要去微软站点上下载这个文件，本把它放入系统System32文件夹。
MySpy是使用C++ Builder6 + Visual C++6 + PhotoShop7 + Microangelo开发的，在关于对话上的Mes'X其实的也是作者Aweay :).
你可以获得任何窗体对应程序的资源图标，而你只需要点击鼠标，图标就保存为真彩色格式、XP风格的格式。
你可以使用窗体Enabled功能来让一些变灰的按钮重新可以使用，这样你可以研究按钮隐藏的秘密或者...
你可以或者屏幕上任何一点的颜色，并可以将颜色表示成为多种颜色格式。
所有的消息保存在MsgData.sys文件中，你可以自己定义这个文件，加入你认识而MySpy不认识的消息。
插件配置文件保存在Plugins.cfg文件中，你可以手动添加插件列表，或者用“；”暂时注释掉不用的插件。
你可以拖动PE文件到MySpy窗体直接查看PE文件信息。
你想得到rm，wmv，Flash的连接地址吗？在Web页，拖动e图标到rm上，打开Run Script功能，加入Templeate后，运行。
Run Script有什么用？这个功能非常强大，如果你对html熟悉的话，这个功能将使你在html里畅游，没有什么可以隐藏的信息。
在Plugins目录下附带了一个插件的例子，如果你对编写插件有兴趣，可以打开它编译成为NetSample.dll，然后修改Plugins.cfg文件加入此插件的引用信息即可。
